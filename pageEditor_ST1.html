Home Page

Technical Description

The Home page was structured using semantic HTML elements such as header, nav, main, and footer to improve readability and maintainability. CSS Flexbox and Grid were used to align elements properly and ensure responsiveness across different screen sizes. A consistent color palette based on blue tones was applied using CSS variables to reflect the SDG 6 theme..

Accessibility

Accessibility was improved by using proper heading hierarchy, alt attributes for images, and ARIA labels for navigation elements. The layout ensures readability with sufficient color contrast and responsive design.

Gallery Page

Technical Description

The gallery page was created using CSS Grid to display images in a responsive layout. Each image is wrapped inside a container with consistent height and uses object-fit: cover for uniform appearance. JavaScript was used to create an interactive modal that displays image details when clicked. Event listeners handle hover effects, modal opening, closing, and customization features such as changing color and font.

Accessibility

Images include descriptive alt text, and interactive elements like buttons are clearly labeled. The modal structure ensures content is readable and accessible. Semantic HTML improves navigation for assistive technologies.

Validation Report | View Live Page

Content Page

Technical Description

The content page uses structured sections and text elements to present information clearly. CSS styling ensures proper spacing, typography, and visual hierarchy. Reusable components from the global stylesheet maintain consistency across the site.

Accessibility

Readable fonts, proper spacing, and semantic HTML elements were used to improve accessibility. Links are clearly identifiable and easy to navigate.

Validation Report | View Live Page

Challenges & Lessons Learned

One major challenge was integrating JavaScript with the gallery modal functionality. This was resolved by properly selecting DOM elements and using event listeners. Another challenge was maintaining consistent styling across pages, which was solved using a shared global CSS file.

Compliance

The website follows good web design practices including accessibility, responsiveness, and semantic HTML usage. Content is structured clearly, and external resources are properly referenced. This aligns with web publishing regulations.
